Font Style

css dapat mengatur style dari teks yang ada pada halaman web yang dibuat. Terdapat beberapa propety yang dapat digunakan untuk mengatur style teks yaitu sebagai berikut.

  • tekt-align: digunakan untuk mengatur peerataan teks, value yang dapat digunakan yaitu left, center, right.
  • color: digunakan untuk memberikan waran pada teks.
  • letter-spacing: mengatur jarak antar karakter.
  • word-spacing: mengatur jarak antar kata.
